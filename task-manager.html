UKSDS Task Manager Setup

This app uses Google Sheets as its database so all employees share one live task list. Complete the one-time setup below — it takes about 5 minutes.

One-time setup (done by admin only):
  1. Go to script.google.com → New Project
  2. Delete existing code. Paste the Code.gs file contents
  3. Click Deploy → New Deployment
  4. Type: Web App | Execute as: Me | Access: Anyone
  5. Click Deploy → Authorize → copy the Web App URL
  6. Paste the URL below and click Connect
